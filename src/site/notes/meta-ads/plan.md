# Meta Ads 从零培训 — 首课计划（2 小时）

**目标**：理论 + 实操结合，结束时在 Meta Ads Manager 里有一个 Sealight 草稿广告，并在 Obsidian 写完笔记。

**账户**：Sealight 实际投放账户（只创建草稿，不发布）

---

## 第一课时结构（120 分钟）

### 模块 1：Meta Ads 核心概念（20 min）
- 广告生态位：Meta Ads 在 DTC 电商里的角色
- 竞价机制：实时竞拍原理（不展开）
- 账户三层结构：Business Manager → Ad Account →人们怎么看广告
- 关键仪表盘位置：Ads Manager / Events Manager / Audiences

### 模块 2：Campaign 架构（20 min）— 实操
- 一起打开 Ads Manager
- 目标选择（Awareness / Traffic / Engagement / Leads / Sales）
- 预算模式：单日预算 vs 总预算
- 实操：创建一个 Sales Campaign 草稿（Sealight）

### 模块 3：Ad Set — 受众与投放（25 min）
- 受众三要素：地区、年龄、性别
- 兴趣定位 vs 行为定位
- Custom Audience（网站访客、客户列表）
- Lookalike Audience（相似受众）
- 版位：Advantage+ 自动 vs 手动
- 实操：为 Sealight 配置第一个 Ad Set

### 模块 4：Ad — 创意与文案（25 min）
- 素材格式：单图、视频、轮播、合集
- 文案结构：Primary Text + Headline + Description + CTA
- 素材规格要求
- 实操：上传/选择素材，写英文文案（结合 sealight-copy-audit）

### 模块 5：Pixel 与追踪（15 min）
- Meta Pixel 是什么
- Conversions API（CAPI）与 Pixel 的关系
- 标准事件：ViewContent / AddToCart / Purchase
- 在 Events Manager 里检查 Pixel 状态

### 模块 6：复盘 + Obsidian 笔记（15 min）
- 回顾今天创建了什么
- 关键概念速查表写入 Obsidian
- 留作业：下节课前阅读 Meta 官方 Blueprint 基础课程
